## Usage

> [!IMPORTANT]
> Make sure your link has web/ at the end! 
> Example: https://yourgithubusername.github.io/yourforksname/web/

### Initial Setup
1. Fork this repo.
2. Enable GitHub Pages: 
   * Go to your fork's settings tab.
   * Click pages in the left sidebar.
   * Select the main branch as your source and click save.
   * Copy your GitHub Pages URL.

---

### Browser Configuration

#### Chromium (Chrome, Edge, Brave)
As Startup Page:
* Click the 3 dots in the top right corner > Settings.
* Click On Startup.
* Select "Open a specific page, or set of pages".
* Paste your URL and save.

As New Tab Page:
* Install the [Custom New Tab URL](https://chromewebstore.google.com/detail/custom-new-tab-url/mmjbdbjnoablegbkcklggeknkfcjkjia) extension.
* Open extension options and check Enabled.
* Paste your URL into the path field and save.

#### Firefox
As Startup Page:
* Go to about:preferences#home.
* Set "Homepage and new windows" to Custom URLs.
* Paste your URL and save.

As New Tab Page:
* Install the [Custom New Tab Page](https://addons.mozilla.org/en-US/firefox/addon/custom-new-tab-page/) extension.
* Go to about:addons > Extensions.
* Select Options for the extension and enter your URL.
