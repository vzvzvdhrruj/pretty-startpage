## Usage

> [!WARNING]
> Make sure your link has web/ at the end! Example: https://yourgithubusername.github.io/yourforksname/web/

### Detailed Deployment
1. Fork this repo:
   Look at the top right of this page and click the fork button to create a copy of this project under your own GitHub account.
2. Enable GitHub Pages: 
   * Go to your fork's settings tab at the top of the page.
   * Click pages in the left sidebar menu.
   * Under the build and deployment section, ensure the source is set to deploy from a branch.
   * Select the main branch as your source and the /root folder, then click save.
   * Wait about 1-2 minutes for GitHub to build the site. A link will appear at the top of the pages section; save your Github Pages URL somewhere.

---

### Detailed Browser Configuration

#### Chromium (Chrome, Edge, Brave, Opera)
Setting as startup/homepage:
* Click the 3 dots in the top right corner of your browser window.
* Click Settings from the dropdown menu.
* Click On Startup in the left-hand navigation list.
* Click the radio button for "Open a specific page, or set of pages".
* Click Add a new page and put in the URL you got, making sure it ends in /web/. (ex: https://yourgithubusername.github.io/yourforksname/web/)
* Save the changes.

Set as new tab page:
* Since Chromium browsers do not allow custom new tab URLs by default, add the Custom New Tab URL extension (https://chromewebstore.google.com/detail/custom-new-tab-url/mmjbdbjnoablegbkcklggeknkfcjkjia).
* Click the 3 dots in the top right corner.
* Click Extensions and then Manage Extensions.
* Find Custom New Tab URL and click Details or Extension Options.
* Check the box labeled Enabled.
* Under the field "URL or local file path" put in your full URL.
* Save and open a new tab to test.

---

#### Firefox
Setting as startup/homepage:
* Click the hamburger menu (three horizontal lines) in the top right corner.
* Click Settings and then select Home on the left side.
* Find the section labeled "Homepage and new windows".
* Click the dropdown menu and set it to Custom URLs.
* Put in the URL you got from GitHub Pages.
* Save and restart the browser.

Set as new tab page:
* Add the Custom New Tab Page extension (https://addons.mozilla.org/en-US/firefox/addon/custom-new-tab-page/).
* Open Add-ons from the Menu (or navigate to about:addons in the address bar).
* Select Extensions in the sidebar.
* Select the Custom New Tab Page extension and click the Options button.
* Enter your full URL in the New Tab URL text box.
* Save and open a new tab to verify it works.
