<h3 align="center">
  # Pretty Startpage

A simple, customizable new tab page/start page for your browser.
Based on the <a href="https://github.com/catppuccin/catppuccin/tree/main/">Catppuccin</a> Mocha color pallete!

---

## Usage

1. Fork this repo.
2. Enable GitHub Pages: 
   * Go to your fork's settings tab.
   * Click pages in the left sidebar.
   * Select the `main` branch as your source and click save.

---

## Customizing

You can adjust basic settings directly within the UI. 

> Tip: You can customize it more by editing the files directly, but I will try to make the settings more advanced later.
